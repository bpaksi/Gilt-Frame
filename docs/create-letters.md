## 1. What to prepare on your Mac

Format everything as **PDF** and put it on a USB drive (FAT or ExFAT). Mac is fine; PDF is the key. [office.fedex](https://www.office.fedex.com/default/faq.html)

Put these files on the USB:

1. **Letters**
   - `letter-1.pdf` – first 1‑page medieval letter.
   - `letter-2.pdf` – second 1‑page letter (or reuse the same file if identical).

2. **Envelope layout**
   - `envelope-front.pdf` with:
     - Top left: your **glyph image only** (no text, no address).
     - Center: the **recipient address**, exactly how USPS expects it (name, street, city/state ZIP). [theletterpilot](https://theletterpilot.com/guides/how-to-address-an-envelope)

No return address text anywhere—that’s intentional for the secret game.

## 2. What to ask the FedEx employee (exact wording)

At the print counter, hand them the USB and say:

1. **For the letters**

> “I have two 1‑page PDFs on this USB that I need printed in color. I’d like them single‑sided on heavier, nicer paper—ideally ivory or a thicker laser stock—so they feel like old letters.” [office.fedex](https://www.office.fedex.com/default/copies.html)

If they ask what paper:

> “Whatever you have that’s heavier than normal copy paper, and preferably not bright white—something suitable for a formal letter.” [fedex](https://www.fedex.com/en-us/small-business/articles-insights/types-of-printing-paper.html)

2. **For the envelopes**

> “I also have an envelope layout PDF on this USB. In the top left, there’s just a symbol—that’s a decorative glyph, not a return address. In the center is the recipient address. Can you print this layout on two standard letter envelopes, exactly as shown, with no return address text anywhere?” [esf](https://www.esf.edu/business/documents/address-letter.pdf)

If they suggest labels instead:

> “If it’s easier, you can print the recipient address twice on labels from this file, and I’ll stick them on the envelopes that have the glyph printed in the top left.” [office.fedex](https://www.office.fedex.com/default/sheet-labels.html)

3. **Mailing without a return address**

When they’re ready to add postage or ship:

> “I don’t want any return address on these at all, just the glyph and the recipient address. These are just regular letters, no special services—I understand they won’t come back to me if undeliverable.” [stampedwithlovexoxo](https://stampedwithlovexoxo.com/blogs/love-letters/can-you-mail-letter-without-return-address)

## 3. Quick checklist before you leave home

- USB drive with:
  - `letter-1.pdf` and `letter-2.pdf`
  - `envelope-front.pdf` (glyph in top left, recipient address centered)
- Recipient address checked for typos.
- Optional: a small note you can hand them:
  - “Color, single‑sided, heavier/ivory paper for letters; 2 envelopes printed with glyph (no return address) and this recipient address; mailed as regular letters.”
